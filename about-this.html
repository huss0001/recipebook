How We Made This Site

Resources & Tools

  • VSCODE
  • TERMINAL
  • NETLIFY
  • GITHUB
  • WHIMSICAL

What We Used Each Resources For:

  • VSCODE - This was used to create html files for each page.
  • TERMINAL - This was used to create a local repository that we could connect to the remote repository on GitHub.
  • NETLIFY - This was used to deploy the website and was connected to GitHub repository so changes could be automatically deployed.
  • GITHUB - This was to store all the project items. We also used GitHub to collaborate and create our own branches which we would push changes to the remote repository to review pull requests. Once the pull request is approved, we would merge the changes with the main branch.
  • WHIMSICAL - This was used to create a visual representation of our layout and structure.
